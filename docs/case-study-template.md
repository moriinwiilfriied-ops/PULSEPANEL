# Template — Mini étude de cas

Structure pour transformer un pilote réussi en mini case study réutilisable. À remplir après une campagne terminée (données depuis le dashboard → détail campagne → Résumé preuve).

---

## 1. Problème

*Quel besoin ou question du client ?*

- [ ] Problème / objectif décrit en 1–2 phrases
- [ ] Contexte (secteur, taille, objectif test)

## 2. Campagne lancée

*Ce qui a été mis en place.*

- **Question** : *(copier depuis le Résumé preuve ou la page détail campagne)*
- **Quota** : …
- **Ciblage** : *(si pertinent)*
- **Durée / période** : …

---

## 3. Résultats

*Chiffres issus du dashboard (Résumé preuve).*

- **Volume** : X réponses / quota Y
- **Rapidité** : temps pour quota *(si disponible)*
- **Qualité** : % valides
- **Coût réel / réponse** : X € *(si disponible)*
- **Chiffre fort retenu** : *(ex. « 100 réponses en 4h », « 92 % de réponses valides »)*

---

## 4. Décision prise

*Qu’est-ce que le client a fait avec les résultats ?*

- [ ] Décision ou action (ex. choix de concept, ajustement produit, validation hypothèse)
- [ ] Suite donnée (relance, V2, autre campagne)
