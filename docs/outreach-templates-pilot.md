# Templates outreach / sales — Pilot PulsePanel

Templates concrets pour DM, email, relances, call et closing. Ton simple, pas de bullshit. Cohérent avec le produit réel.

---

## 1. DM LinkedIn court

**Objet** : premier contact, court.

---
Salut [Prénom],

Je vois que [vous travaillez sur X / vous lancez Y]. Vous faites des tests (packaging, prix, slogan) pour valider avant de lancer ?

On a un outil pilot qui permet de lancer une campagne de test en quelques minutes et d’avoir des réponses réelles en 24–72 h (dashboard + export). Pas une grosse étude, juste un premier test propre.

Si ça peut vous intéresser, on peut en parler 15 min. Sinon pas de souci.

[Prénom]

## 5. Script call découverte 15 min

**Objectif** : Qualifier le besoin, voir si le pilot correspond, booker démo si oui.

| Étape | Durée | Contenu |
|-------|--------|---------|
| Intro | 1 min | Présentation courte : PulsePanel = outil pour lancer un test (campagne question + options), réponses réelles en 24–72 h. |
| Besoin | 5 min | « Qu’est-ce que vous aimeriez tester en ce moment ? » (packaging, prix, slogan, concept). « Vous avez déjà fait des tests ? Comment ? » |
| Fit | 4 min | Expliquer le pilot : dashboard, création campagne, diffusion app, export + résumé. « Ce qui ne va pas : pas un panel France représentatif, pas un rapport 20 pages. Un test rapide, propre. » |
| Suite | 3 min | Si intéressé : « Je vous envoie un récap + une proposition pilot par email. On peut faire une démo 20 min pour vous montrer le dashboard. » Si pas prêt : « Pas de souci. On peut reprendre quand vous avez un besoin concret. » |
| Clôture | 2 min | Confirmer next step (email récap, démo ou pas), remercier. |

**À ne pas faire** : Promettre des fonctionnalités non prêtes. Parler plus de 15 min si le prospect n’est pas qualifié.

## 6. Script démo 20 min orientée pilot

**Objectif** : Montrer le produit réel, ancrer la promesse « test en 24–72 h ».

| Étape | Durée | Contenu |
|-------|--------|---------|
| Contexte | 2 min | « Je vous montre le dashboard tel que vous l’aurez. On ne fait pas de slides : on est sur l’outil. » |
| Login / org | 2 min | Login, sélection d’org (si multi-org). Montrer la home : campagnes, crédit, KPI simples. |
| Création campagne | 6 min | Nouvelle campagne : nom, question, options (ex. 2 options type A/B), quota, récompense. Montrer l’estimation de coût. « Vous validez, vous activez : le crédit est débité, la campagne part. » |
| Suivi / résultat | 5 min | Ouvrir une campagne (seed ou existante). Montrer : réponses en direct, progression quota, qualité (valides / trop rapide). Puis détail : export CSV, bloc « Résumé preuve » (chiffre fort, copier le résumé). « À la fin vous avez les données + un résumé exploitable. » |
| Billing | 2 min | Billing : solde, recharge Stripe, ledger. « Vous rechargez quand vous voulez, vous consommez au lancement des campagnes. » |
| Closing | 3 min | « Pour le pilot on fait quoi : on vous crée l’accès (ou vous utilisez une org dédiée), on définit ensemble la première campagne, vous lancez. Premier résultat sous 72 h. Ensuite on regarde le résultat et on voit si vous voulez une V2 ou une autre campagne. » Proposer offre (prix, CTA). |

**Référence** : `docs/demo-script-10min.md` pour le détail des écrans.

## 7. Script closing pilot

**Objectif** : Obtenir l’accord pour le pilot (email, devis, paiement selon votre process).

---
« En résumé, le pilot c’est :

- Accès dashboard + support pour la première campagne.
- Vous choisissez la question et le quota, on valide ensemble.
- Vous lancez, vous avez le résultat en 24–72 h.
- Vous récupérez l’export des réponses et le résumé preuve. Si vous voulez une V2 (relancer un test), vous pouvez dupliquer la campagne en un clic.

[Prix selon votre positionnement : ex. forfait 299–499 € HT pour une première campagne, ou crédit consommable 100–300 €.]

Prochaine étape : je vous envoie [un récap par email / un devis]. Vous validez, on fixe la date d’onboarding et on lance la première campagne. »
---

**Si objection budget** : « On peut commencer sur un petit quota (ex. 50 réponses) pour que le coût reste maîtrisé. L’idée est de voir si le format vous convient. »

**Si objection timing** : « On peut booker l’onboarding dès que vous êtes prêt. Comptez 30 min pour être opérationnel. »
